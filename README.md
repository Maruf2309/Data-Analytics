# Data-Analytics

Project Overview
This project presents an interactive HR Analytics Dashboard built in Excel. It provides insights into employee distribution, salary trends, hiring patterns, and workforce composition using Pivot Tables, Power Query, and visualizations.

2. Objectives
- Analyze employee distribution across departments
- Compare Permanent vs Probation employees
- Track hiring trends over time
- Evaluate salary distribution and averages
- Understand gender diversity

3. Tools Used
- Microsoft Excel
- Power Query (Data Cleaning & Transformation)
- Pivot Tables & Pivot Charts
- Slicers for interactivity

4. Key Features
- KPI Cards: Total Employees, Total Salary, Active Employees, Average Salary
- Department-wise employee distribution (Bar Chart)
- Gender and Employee Type distribution (Pie/Donut Charts)
- Average salary by designation
- Monthly hiring trend (Line Chart)
- Interactive filters using slicers

5. Dataset Description
### Dataset
- <a href="https://github.com/Maruf2309/Data-Analytics/blob/main/HRDataset.xlsx">Dataset View </a>
The dataset includes employee-level information such as Name, Department, Designation, Employee Type, Date of Join, Gender, Salary, Working Status, and more.
7. Steps Performed
1. Cleaned data using Power Query
2. Converted date fields and created Month/Year columns
3. Built Pivot Tables for aggregation
4. 4. Created charts for visualization
5. Added slicers for dynamic filtering
6. Designed dashboard layout

7. Insights
- Majority of employees belong to specific departments
- Salary varies across designations
- Hiring trends show variation across months
